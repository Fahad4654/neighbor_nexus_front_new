# **App Name**: Neighbor Nexus

## Core Features:

- User Verification: Verify users through email and address confirmation to build trust within the community.
- Transaction Management: Facilitate and track tool rentals and skill service exchanges.
- Local Zone Definition: Allow users to define and join a 'Nexus' zone to foster local engagement.
- Commission Tracking: Automatically calculate and track commissions from successful exchanges.
- KPI Dashboard: Display Activation Rate, Transaction Volume, and Local Penetration metrics.
- CAC/CLV Analysis: Provide insights into Customer Acquisition Cost (CAC) and Customer Lifetime Value (CLV) ratio.
- Rating & Review System: Enable users to rate and review tools or services used, to increase confidence and community.

## Style Guidelines:

- Primary color: A vibrant, friendly blue (#5089C6) evokes trust and connectivity.
- Background color: A light blue (#EBF4FA), very close in hue to the primary color but lightly desaturated and bright, provides a clean and open feel.
- Accent color: A soft teal (#4DB6AC) provides a sense of calm and trustworthiness, drawing attention to key actions and elements.
- Body and headline font: 'PT Sans', a humanist sans-serif to maintain a balance of modernity and warmth, enhancing readability.
- Use simple, recognizable icons to represent different tools, services, and community actions.
- Emphasize clear sections for listings, user profiles, and community zones for easy navigation.
- Employ subtle animations for transitions and feedback, like highlighting successful transactions or updates.
